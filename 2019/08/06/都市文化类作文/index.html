第4讲 都市漫步 城市文化类记叙文写作

城市文化

  • 审题
    • 城市
      • 选择自己去过的城市
      • 选择旅行去过的城市
    • 文化
      • 饮食文化 - 戏曲文化
      • 人文景观 - 历史名人
      • 民风民俗 - 手工艺品
      • 神话传说 - 历史故事
      • 地域建筑
    • 立意
      • 表达对自己生活城市的热爱
      • 表达对城市拥有文化的自豪
      • 欢迎外人来自己城市游玩
      • 揭示所在城市的文化底蕴

方法总结

  1. 将日常所学的诗词，要恰如其分地运用到作文中，来增加作文的文采和增强作文的文化底蕴
  2. 写作文的开头，可以运用对比映衬法，来体现对所写地点的热爱之情

如何摘抄

  1. 读 - 读对作文有益的文章
    1. 《满分作文选》
    2. 木头哥的原创范文
    3. 木头哥的摘抄分享
    4. 木头哥分享的学生优秀范文
  2. 摘 - 摘取含出色修辞和描写的句子
  3. 抄 - 把优秀的语句超载本子上
  4. 背 - 有用的摘抄都要背下来
  5. 用 - 在考场作文中插花
  6. 造 - 优秀语句融会贯通，创造佳句

方法总结

  1. 描写所写地点，一定要抓住这个地区最具代表性的事物

方法总结

  1. 写城市文化类作文，一定要写这个城市的历史
  2. 叙述历史时，可以用排比的修辞展示，既简洁，又能展现文章的气势

方法总结

  1. 描写食物时，要调动多种感官：色，香，味

方法总结

  1. 城市文化类作文的结尾，要呼应文章题目，要抒发自己对城市的情感

模板总结

  • 第一部分(1段)
    • 引用诗词，运用对比，引出所写城市
    • 呼应文章题目
  • 第二部分(2、3段)
    • __，我爱你的壮美山海，爱你的英姿风华
    • 抓住城市的代表性事物，来书写城市的沧桑巨变
  • 第三部分(4、5段)
    • __，我也爱你历经沧桑后的沉静恬淡，爱你洗尽铅华后的天然诚挚
    • 根据城市经历的历史，来书写城市的沧桑巨变
  • 第四部分(6、7段)
    • __，我更爱你悠悠岁月里的寸寸柔情，爱你平凡生活中的人间百味
    • 抓住城市的特色美食，来书写城市的人事百味
  • 第五部分(8段)
    • 呼应文章题目
    • 抒发对所写地点的热爱之情
